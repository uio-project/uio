## Agent: repo-stats

This agent provides various statistics about the repository it operates on.

### Capabilities:

*   Count lines of code per file type.
*   List the top 5 largest files in the repository.
*   Report the number of unique Git contributors.

### Usage:

To use this agent, simply invoke it without any specific arguments. It will automatically gather and report the relevant statistics for the current repository.
